## Engineering Shell Frontends-ts

### Install

```
npm install

# 重新安装依赖并强制编译
npm install --force --build-from-source
```

### Usage

#### Run the app

```
npm run start
```

#### Build the app (manual)

```
npm run build
```

#### Build the app

```
npm run package
```

#### Build the app for install

```
npm run dist
```
