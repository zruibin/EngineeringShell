## Engineering Shell Frontends

### Install

```
npm install
```

### Usage

#### Run the app

```
npm run start
```

#### Build the app (manual)

```
npm run build
```

#### Build the app

```
npm run package
```


#### Build the app for install

```
npm run installer
```
