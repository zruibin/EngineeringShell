## Engineering Shell Frontends-tauri

### Install

```
npm install

# 重新安装依赖并强制编译
npm install --force --build-from-source
```

### Usage

#### Build the front

```
npm run start
```

#### Build the front

```
npm run build
```

#### Run the app

```
npm run dev
```

#### Build the app

```
npm run prod
```

#### Run and Build on iOS/Android

* Init 

```
android：
npm tauri android init

ios：
npm tauri ios init
```

```
npm run dev:ios
npm run prod:ios
npm run dev:android
npm run prod:android
```
